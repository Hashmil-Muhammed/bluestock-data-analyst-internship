D1

Orientation & Python Foundation

Environment setup · Python basics · OOP · Target: CSV stats script
▼
🕘Morning (9 AM – 11 AM) — Log into workspace & mark Attendance. Read the full Prerequisite Guide PDF.
🔧Setup (11 AM – 12 PM) — Install pgAdmin 4 (you already have VS Code + Jupyter). Create a fresh GitHub repo: bluestock-internship. Activate Google Colab account.
Resource: youtube.com/@TraversyMedia → PostgreSQL Crash Course
🐍Python Revision (1 PM – 4 PM) — Variables, data types, lists, dicts, loops, functions, file I/O, error handling. Then OOP: classes, objects, inheritance, dunder methods.
Resource: youtube.com/@coreyms (Corey Schafer Python series) · docs.python.org
🎯Target Task (4 PM – 6 PM) — Write a Python script (no Pandas) that reads a financial CSV from Kaggle and prints: row count, column names, min/max/mean of numeric columns.
Dataset: Kaggle → "Nifty 50 Stock Market Data" or any financial CSV
📝Standup (6 PM) — Log in workspace: "Completed Python/OOP revision, pgAdmin setup, wrote CSV stats script."
D2

NumPy, Pandas & Data Cleaning

Arrays · DataFrames · EDA · Target: Complete 1 EDA notebook on financial data
▼
🕘Morning (9 AM – 9:15 AM) — Mark Attendance.
📐NumPy (9:15 AM – 11 AM) — Arrays, operations, indexing, slicing, broadcasting, random, reshaping, math operations.
Resource: youtube.com/@KeithGalli · numpy.org/doc
🐼Pandas (11 AM – 2 PM) — DataFrame, Series, read_csv, head/tail, info, describe. Then: loc/iloc, filtering, groupby, merge/join, pivot, apply, fillna, dropna.
Resource: Corey Schafer Pandas Series · pandas.pydata.org
🧹Data Cleaning practice (2 PM – 4 PM) — Download a Kaggle financial dataset. Handle: missing values, duplicate rows, type conversion, outlier detection, column renaming.
Resource: youtube.com/@KeithGalli · kaggle.com/learn (Pandas course, free cert)
🎯Target Task (4 PM – 6 PM) — Complete a full EDA Jupyter Notebook: shape, describe, missing %, correlation heatmap, distribution plots, top/bottom 5 rows. Commit to GitHub.
Platform: Jupyter Notebook or Google Colab
📝Standup (6 PM) — "Completed NumPy + Pandas revision, finished EDA notebook on [dataset name], pushed to GitHub."
D3

SQL + PostgreSQL + Capstone Assigned ⚡

SQL fundamentals · pgAdmin · CTEs · Window functions · Capstone planning starts today
▼
🕘Morning (9 AM – 9:15 AM) — Mark Attendance. Check workspace for Capstone project assignment notification.
🗄️SQL Basics (9:15 AM – 12 PM) — SELECT, WHERE, ORDER BY, GROUP BY, HAVING, LIMIT, DISTINCT, aliases. Then: INNER/LEFT/RIGHT/FULL JOINs, subqueries, CTEs (WITH clause).
Resource: youtube.com/@freecodecamp (SQL Full Course) · w3schools.com/sql · postgresqltutorial.com
🐘PostgreSQL Setup (12 PM – 2 PM) — Open pgAdmin 4. Create a new database, import a financial CSV. Run basic queries. Explore SERIAL, JSONB, indexes, psycopg2 basics.
Resource: youtube.com/@TraversyMedia PostgreSQL · postgresql.org/docs
🪟Advanced SQL (2 PM – 4 PM) — Window functions: ROW_NUMBER, RANK, LAG, LEAD, PARTITION BY. Data cleaning in SQL: CAST, COALESCE, NULLIF, CASE WHEN, string + date functions.
Resource: youtube.com/@AlexTheAnalyst · mode.com/sql-tutorial
🎯Target + Capstone (4 PM – 6 PM) — Run 10+ SQL queries on financial data in PostgreSQL. Plan capstone: set up Google Drive folder structure (Source Code / Datasets / Documentation / PPT / Demo Video). Start README.
Important: Capstone is assigned today — start immediately!
📝Standup (6 PM) — "Completed SQL + PostgreSQL setup. Ran 10 queries. Capstone received and folder structure created."

Visualisation + Flask + Capstone Work

5+ chart types · REST API · psycopg2 · First capstone insights
▼
🕘Morning (9 AM – 9:15 AM) — Mark Attendance.
📊Matplotlib & Seaborn (9:15 AM – 11:30 AM) — Line, bar, scatter, histogram, boxplot, heatmap. Styling, subplots, sns themes, annotations, choosing the right chart for financial data.

📝Standup (6 PM) — "Completed Matplotlib/Seaborn + Flask API with PostgreSQL. Capstone EDA started, first insights ready."
D5

Fintech Domain + ML Basics + Capstone 50%

Stock market · Mutual funds · yfinance · Linear regression · Capstone 50%+ complete
▼
🕘Morning (9 AM – 9:15 AM) — Mark Attendance.
📈Stock Market (9:15 AM – 11 AM) — What is a stock, equity, NSE/BSE, IPO. OHLC data, market cap, trading volume, bid-ask spread. NIFTY 50, SENSEX. P/E, EPS, P/B, balance sheet basics. Fetch OHLC data using yfinance in Python.
Resource: youtube.com/@CARachanaRanade · zerodha.com/varsity · pypi.org/project/yfinance
💰Mutual Funds (11 AM – 12:30 PM) — NAV, AUM, AMC, expense ratio, exit load. Types: equity/debt/hybrid/ELSS/index. CAGR, XIRR, Sharpe ratio, alpha/beta. SIP vs lump sum. Calculate CAGR in Pandas.
Resource: youtube.com/@CARachanaRanade MF Series · amfiindia.com · zerodha.com/varsity
🤖ML Basics (12:30 PM – 2 PM) — Supervised vs unsupervised. Linear regression: R², RMSE, MAE. Logistic regression, confusion matrix, precision/recall/F1. sklearn: fit(), predict(), train_test_split, pipeline.
Resource: youtube.com/@statquest · youtube.com/@KrishNaik · kaggle.com/learn
🎯Capstone (2 PM – 6 PM) — Complete data analysis section of capstone. Begin modelling (linear regression on financial dataset). Push all work to GitHub. Target: Capstone 50%+ complete.
Goal: Fintech domain notes ready + capstone halfway done
📝Standup (6 PM) — "Completed Fintech domain learning (stocks + MF + ML basics). Capstone analysis section done — 50%+ complete."
